In this, I've used the software Rhino, more specifically the addin Grasshopper. The Grasshopper addin has 2 packages available for 
carrying out solar analysis : Ladybug and Honeybee
Using these, I've built a code which can calculate the radiation intensity on desired surfaces of Rhino objects based on the sunlight data
(obtained from .epw file)
